Usage instructions: frclinks.com +

Command

Short form

Operand(s)

Go to the...
/team /t   list of all registered FRC teams.
/team/ /t/ team number FIRST information page for the given team.
/team/ /t/ area id/[year] team list for the given area ID (e.g. CA-USA, ON-Canada).
/website/ /w/ team number website of the given team.
/tba/ /tba/ team number Blue Alliance page for the given team.
/cdm/ /cdm/ team number Chief Delphi Media page for photos tagged with the given team.
       
/event/ /e/ event id/[year] team list for the given event ID (see below for codes).
/event/schedule/ /e/s/ event id/[year] qualification match schedule for the given event ID (see below for codes).
/event/matchresults/ /e/m/ event id/[year] qualification match results for the given event ID (see below for codes).
/event/rankings/ /e/r/ event id/[year] rankings for the given event ID (see below for codes).
/event/awards/ /e/a/ event id/[year] awards for the given event ID (see below for codes).
/event/tba/ /e/tba/ event id/[year] Blue Alliance page for the given event ID (see below for codes).
/championship /c /[year] Championship Event page.
/regionals /r /[year] Regional Events page.
       
/controlsystem /cs   2009-present Control System page.
